# 02-初始化工具与技能

> 安装 OpenClaw CLI 和基础技能，构建你的 AI 助理工具链

## 📦 安装 OpenClaw CLI

### 方式一：Homebrew (推荐)

**macOS:**

```bash
brew install openclaw
openclaw --version
```

**验证：**

```bash
$ openclaw --version
OpenClaw 2026.4.2 (d74a122)
```

### 方式二：pip 安装

```bash
pip install openclaw-cli
```

### 方式三：源码安装

```bash
git clone https://github.com/openclaw/openclaw
cd openclaw
pip install -e .
```

## 🎯 安装 clawhub CLI

**clawhub** 是 OpenClaw 的技能管理工具，用于安装、更新、发布技能。

```bash
# 已包含在 openclaw-cli 中
clawhub --cli-version
```

**预期输出：**

```
0.9.0
```

## 🔐 配置 clawhub

### 1. 登录 clawhub

```bash
clawhub login
```

**步骤：**
1. 浏览器会打开 clawhub.ai 登录页面
2. 授权后复制 token
3. 粘贴到 CLI 完成登录

**预期输出：**

```
- Checking token
✔ kekolike-sudo
```

### 2. 验证登录状态

```bash
clawhub whoami
```

## 📚 安装基础技能

### 技能分类说明

| 类别 | 用途 | 示例 |
|------|------|------|
| **基础工具** | 通用工具 | `summarize`, `oracle` |
| **开发工具** | 代码相关 | `github` |
| **办公工具** | 工作自动化 | `gog` |
| **系统工具** | 本地控制 | `weather`, `video-frames` |

### 推荐安装列表

**必装技能：**

```bash
# Google Workspace - 邮件、日历、文档管理
clawhub install gog

# GitHub - 代码仓库管理
clawhub install github

# 内容摘要 - 网页/视频转文字摘要
clawhub install summarize

# Oracle - 提示工程工具
clawhub install oracle
```

**可选技能（按需）：**

```bash
# 天气查询
clawhub install weather

# Sonos 音响控制
clawhub install sonoscli

# 视频帧提取
clawhub install video-frames
```

### 验证安装

```bash
clawhub list
```

**预期输出：**

```
gog  1.0.0
github  1.0.0
summarize  1.0.0
oracle  1.0.0
```

## ⚙️ 配置技能

### 1. Google Workspace (gog)

**用途：** Gmail、Google Calendar、Google Drive 集成

**配置步骤：**

```bash
# 1. 生成 Google Cloud API Key
# https://console.cloud.google.com/apis/credentials

# 2. 设置环境变量
export GOOGLE_APPLICATION_CREDENTIALS="/path/to/your-credentials.json"

# 3. 测试连接
gog --test
```

### 2. GitHub (github)

**用途：** GitHub API 集成，Issue、PR 管理

**配置步骤：**

```bash
# GitHub CLI 已安装则自动使用
gh auth status

# 或使用 GitHub Token
export GITHUB_TOKEN=ghp_your_token_here
```

### 测试 github

```bash

```

### 测试 summarize

```bash
# 摘要一个网页
summarize https://example.com
```

## 📝 最佳实践

### 1. 技能安装顺序

1. **基础工具** - `summarize`, `oracle`
2. **开发工具** - `github`
3. **办公工具** - `gog`
4. **领域工具** - 按需安装

### 2. 技能管理

```bash
# 查看已安装技能
clawhub list

# 更新所有技能
clawhub update

# 安装特定版本
clawhub install github --version 1.0.0

# 卸载技能
clawhub uninstall github
```

### 3. 安全原则

- ⚠️ **不安装可疑技能** - 查看 VirusTotal 评分
- ⚠️ **最小权限原则** - 只授予必要权限
- ⚠️ **定期审计** - 检查已安装技能的来源
