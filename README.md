## 🛠️ 环境要求

**最低配置：**
- **操作系统：** macOS 12+ / Linux (Ubuntu 20.04+)
- **内存：** 8GB RAM
- **磁盘：** 50GB 可用空间
- **Python：** 3.10+

**推荐配置：**
- **CPU：** M 系列芯片 / 8 核+
- **内存：** 16GB+
- **磁盘：** 100GB+ SSD

## 🚀 快速开始

```bash
# 1. 克隆本仓库
git clone https://github.com/kekolike-sudo/Super-Individual
cd Super-Individual

# 2. 检查环境
./scripts/check-env.sh

# 3. 一键部署
./scripts/deploy.sh

# 4. 安装基础技能
clawhub install gog github summarize
```

## 🛡️ 安全原则

- ✅ **本地优先** - 敏感数据不上传
- ✅ **开源透明** - 所有脚本开源可审计
- ✅ **最小权限** - 只授予必要 API 权限
- ✅ **定期审计** - 检查已安装技能和权限

## 📝 避坑指南

**常见陷阱：**

1. **❌ 盲目安装技能**
   - 后果：安全风险、性能下降
   - ✅ 正确做法：先查看 VirusTotal 评分

2. **❌ 过度依赖云服务**
   - 后果：数据泄露、成本失控
   - ✅ 正确做法：优先本地部署方案

3. **❌ 忽略性能优化**
   - 后果：响应慢、资源浪费
   - ✅ 正确做法：定期执行性能检查

4. **❌ 未配置日志**
   - 后果：问题难以排查
   - ✅ 正确做法：统一日志规范
