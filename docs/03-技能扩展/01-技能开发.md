# 03-技能开发实战

> 从零开始创建你的第一个 OpenClaw 技能，掌握核心开发流程

## 🎯 技能是什么？

**OpenClaw 技能** 是一个独立的 Python 脚本，封装特定功能，通过 CLI 工具暴露给 AI 调用。

### 技能结构

```
my-skill/
├── SKILL.md          # 技能描述和用法
├── skill.py          # 主脚本
├── README.md         # 详细说明（可选）
└── tests/            # 测试用例（可选）
```

## 🛠️ 创建第一个技能

### 1. 创建技能目录

```bash
cd /path/to/skills
mkdir -p my-first-skill
cd my-first-skill
```

### 2. 创建 SKILL.md

```markdown
# My First Skill

一个示例技能，演示 OpenClaw 技能的基本结构。

## 用途

- 输出欢迎消息
- 执行简单计算
- 演示技能调用流程

## 用法

```bash
# 基本调用
my-first-skill hello

# 带参数
my-first-skill calculate 10 + 20 * 3
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `command` | string | 是 | 命令类型：`hello` 或 `calculate` |
| `expression` | string | 否 | 数学表达式（calculate 命令） |

## 示例

```bash
$ my-first-skill hello
Hello, OpenClaw!

$ my-first-skill calculate 10 + 20 * 3
40
```
```

### 3. 创建 skill.py

```python
#!/usr/bin/env python3
"""My First Skill - OpenClaw 技能示例"""

import argparse
import sys

def main():
    """主函数"""
    parser = argparse.ArgumentParser(
        description="My First Skill - OpenClaw 技能示例"
    )
    parser.add_argument(
        "command",
        choices=["hello", "calculate"],
        help="命令类型"
    )
    parser.add_argument(
        "expression",
        nargs="?",
        default=None,
        help="数学表达式（calculate 命令）"
    )
    
    args = parser.parse_args()
    
    if args.command == "hello":
        print("Hello, OpenClaw!")
        
    elif args.command == "calculate":
        if not args.expression:
            print("Error: 需要表达式参数", file=sys.stderr)
            sys.exit(1)
            
        try:
            result = eval(args.expression)
            print(result)
        except Exception as e:
            print(f"Error: {e}", file=sys.stderr)
            sys.exit(1)

if __name__ == "__main__":
    main()
```

## 🔧 技能测试

### 单元测试

```python
# tests/test_skill.py
import unittest
from skill import calculate_expression

class TestSkill(unittest.TestCase):
    def test_addition(self):
        self.assertEqual(calculate_expression("2 + 3"), 5)
    
    def test_multiplication(self):
        self.assertEqual(calculate_expression("2 * 3"), 6)
    
    def test_invalid_expression(self):
        with self.assertRaises(Exception):
            calculate_expression("invalid")

if __name__ == "__main__":
    unittest.main()
```

**运行测试:**

```bash
python -m unittest tests/test_skill.py
```
