# 05-基础命令速查

## 章节内容

### 5.1 状态诊断

#### `openclaw status` - 系统状态
- CPU 使用率
- 内存占用
- 磁盘空间
- 服务状态

#### `openclaw logs` - 日志查看
- 最近日志
- 过滤选项
- 日志级别

#### `openclaw doctor` - 健康检查
- 依赖检查
- 配置验证
- 问题诊断

### 5.2 会话管理

#### `openclaw sessions list` - 查看历史
- 列出所有会话
- 会话详情
- 会话状态

#### `openclaw sessions cleanup` - 清理过期
- 自动清理
- 手动清理
- 保留策略

#### `openclaw session kill` - 强制终止
- 终止指定会话
- 清理资源
- 状态恢复

### 5.3 工具管理

#### `openclaw tools list` - 工具列表
- 已安装工具
- 工具状态
- 版本信息

#### `openclaw tools status` - 工具状态
- 详细状态
- 性能指标
- 错误信息

#### `openclaw tools reset` - 工具重置
- 重置工具
- 恢复默认配置
- 清理缓存
