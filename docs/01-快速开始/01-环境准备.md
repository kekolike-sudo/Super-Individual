# 01-环境准备

> 准备工作：检查并配置运行环境

## 📋 环境要求

### 系统要求

| 项目 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **操作系统** | macOS 12 / Ubuntu 20.04 | macOS 14+ / Ubuntu 22.04+ |
| **CPU** | 4 核 | 8 核+ (M 系列芯片优先) |
| **内存** | 8GB | 16GB+ |
| **磁盘** | 50GB 可用空间 | 100GB+ SSD |

### 软件依赖

- ✅ **Python 3.10+** - AI 脚本运行环境
- ✅ **Node.js v20+** - clawhub 工具依赖
- ✅ **git 2.30+** - 版本控制
- ✅ **jq** - JSON 处理工具

## 🔍 环境检查

运行以下命令检查环境：

```bash
# 检查系统信息
uname -a

# 检查 Python 版本
python3 --version

# 检查 Node.js 版本
node --version

# 检查 Git 版本
git --version

# 检查 jq
jq --version
```

**预期输出示例：**

```
Darwin your-mac.local 23.4.0 Darwin Kernel Version 23.4.0
Python 3.11.5
v20.10.0
git version 2.40.1
jq-1.6
```

## 🛠️ 工具安装

### 1. Python 环境

**macOS (Homebrew):**

```bash
brew install python@3.11
```

**Ubuntu:**

```bash
sudo apt update
sudo apt install python3.11 python3-pip python3-venv
```

### 2. Node.js

**macOS (Homebrew):**

```bash
brew install node@20
```

**Ubuntu:**

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### 3. jq 工具

**macOS:**

```bash
brew install jq
```

**Ubuntu:**

```bash
sudo apt install jq
```

### 4. OpenSSL (用于某些 CLI 工具)

**macOS:**

```bash
brew install openssl
```

**Ubuntu:**

```bash
sudo apt install openssl
```

## 🧪 验证安装

运行测试脚本：

```bash
#!/bin/bash
# check-env.sh

echo "=== OpenClaw 环境检查 ==="
echo ""

echo "1. Python 版本:"
python3 --version

echo ""
echo "2. Node.js 版本:"
node --version

echo ""
echo "3. Git 版本:"
git --version

echo ""
echo "4. jq 版本:"
jq --version

echo ""
echo "5. 环境变量:"
echo "   PATH: $PATH"

echo ""
echo "=== 检查完成 ==="
```

运行：

```bash
chmod +x check-env.sh
./check-env.sh
```
