# 02-部署方式对比

> 选择适合你的 OpenClaw 部署方案

## 📊 部署方式一览

| 部署方式 | 难度 | 成本 | 可控性 | 适合人群 |
|------|--|------|------|------|
| **本地 Homebrew** | ⭐⭐ | 免费 | ✅ 100% | 开发者 |
| **宝塔面板** | ⭐ | 免费 | ✅ 90% | 小白用户 |
| **JVSClaw** | ⭐⭐⭐ | 付费 | ✅ 95% | 企业级 |
| **Docker** | ⭐⭐ | 免费 | ✅ 90% | 技术人员 |
| **云服务器** | ⭐⭐⭐ | 付费 | ✅ 80% | 需要远程访问 |

## 🏠 方案一：本地 Homebrew 安装（推荐）

**适用人群：** 开发者、技术爱好者、隐私敏感用户

**优点：**
- ✅ **完全免费** - 开源社区驱动
- ✅ **本地部署** - 数据完全可控
- ✅ **快速迭代** - 紧跟 OpenClaw 更新
- ✅ **无需付费** - 无订阅成本

**缺点：**
- ⚠️ 需要命令行基础
- ⚠️ 本地资源占用
- ⚠️ 无法远程访问（需自建 SSH）

**安装步骤：**

```bash
# 1. 检查依赖
brew update

# 2. 安装 OpenClaw CLI
brew install openclaw

# 3. 验证安装
openclaw --version

# 4. 配置 GitHub
gh auth login

# 5. 登录 clawhub
clawhub login
```

**配置建议：**

```bash
# GPG 签名（可选，推荐）
git config --global commit.gpgsign true
git config --global user.signingkey YOUR_KEY_ID

# 环境变量
export OPENCLAW_WORKDIR="$HOME/.openclaw/workspace"
```

## 🖥️ 方案二：宝塔面板

**适用人群：** 小白用户、需要可视化界面

**优点：**
- ✅ **可视化界面** - 无需命令
- ✅ **一键部署** - 3 分钟搞定
- ✅ **免费插件** - 无需付费
- ✅ **面板管理** - 方便维护

**缺点：**
- ⚠️ 面板安全性风险
- ⚠️ 部分功能受限
- ⚠️ 依赖宝塔生态

**安装步骤：**

1. 访问宝塔官网安装面板
2. 进入应用商店 → 搜索 OpenClaw
3. 点击安装 → 自动配置
4. 面板启动 → 访问 Web UI

**注意事项：**
- ⚠️ 不要使用默认密码
- ⚠️ 定期更新面板
- ⚠️ 配置防火墙

## 🏢 方案三：JVSClaw

**适用人群：** 企业级用户、高安全需求

**优点：**
- ✅ **端到端加密** - 数据隐私保护
- ✅ **6 核 12GB** - 性能强劲
- ✅ **14 天免费** - 先试后买
- ✅ **邀请码机制** - 安全可控

**缺点：**
- ⚠️ 需要邀请码
- ⚠️ 付费订阅
- ⚠️ 学习成本

**获取邀请码：**
1. 访问 JVSClaw 官网
2. 提交申请（企业邮箱优先）
3. 审核通过 → 收到邀请码
4. 激活 → 开始使用

## 🐳 方案四：Docker 容器化

**适用人群：** 技术人员、需要隔离环境

**优点：**
- ✅ **环境隔离** - 不影响主机
- ✅ **快速部署** - 一键启动
- ✅ **易于备份** - 镜像管理
- ✅ **跨平台** - Linux/macOS/Windows

**缺点：**
- ⚠️ 需要 Docker 基础
- ⚠️ 资源占用
- ⚠️ 网络配置复杂

**安装步骤：**

```bash
# 1. 拉取镜像
docker pull openclaw/openclaw:latest

# 2. 启动容器
docker run -d \
  --name openclaw \
  -v $HOME/.openclaw:/openclaw \
  -p 8080:8080 \
  openclaw/openclaw

# 3. 访问 Web UI
open http://localhost:8080
```

## ☁️ 方案五：云服务器

**适用人群：** 需要远程访问、团队协作

**优点：**
- ✅ **7×24 在线** - 持续服务
- ✅ **远程访问** - 随时随地
- ✅ **资源弹性** - 按需扩展
- ✅ **团队协作** - 多用户支持

**缺点：**
- ⚠️ **持续费用** - 每月 $5-50
- ⚠️ **配置复杂** - SSH/API 需要配置
- ⚠️ **安全风险** - 公网暴露

**推荐云服务商：**
- **AWS EC2** - 稳定性高
- **DigitalOcean** - 性价比高
- **阿里云** - 国内访问快

**部署脚本：**

```bash
#!/bin/bash
# deploy-on-server.sh

# 1. 更新系统
sudo apt update && sudo apt upgrade -y

# 2. 安装依赖
sudo apt install python3.11 python3-pip nodejs git jq

# 3. 安装 OpenClaw
pip install openclaw-cli

# 4. 配置防火墙
sudo ufw allow 8080/tcp
sudo ufw enable

# 5. 设置开机自启
systemctl enable openclaw
systemctl start openclaw
```
